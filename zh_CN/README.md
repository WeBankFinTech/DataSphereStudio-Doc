## 引言

DataSphere Studio（简称 DSS）是微众银行自研的数据应用开发管理集成框架。

在统一的UI下，DataSphere Studio 以工作流式的图形化拖拽开发体验，将满足从数据交换、脱敏清洗、分析挖掘、质量检测、可视化展现、定时调度到数据输出应用等，数据应用开发全流程场景需求。
